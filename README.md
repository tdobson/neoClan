# neoClan: Club Attendance Management

neoClan is an intuitive checkin and attendance management system crafted to track and manage the attendance of sports club members at events. 

Transitioning from Google Sheets to a modern React application, neoClan offers a user-friendly web interface accessible across devices.

![neoClan Logo](https://i.imgur.com/vSCuFBc.png)

**Last updated: 03 May 2023**

## Overview

- **Universal Application**: Designed to cater to adventure sports clubs, from The Climbing Clan and The Caving Crew to any other organization.
- **User-Friendly Interface**: Presents a streamlined interface allowing club managers and officials to check members in upon arrival and monitor various stages of arrivals.
- **Modern Tech Stack**: Built with the powerful [React](https://reactjs.org/) library and aesthetically designed with [Material-UI](https://material-ui.com/).

## Prerequisites

You'll want to have node.js installed

## Installation & Setup

Run

 npm -i 

to install.

## Integration

- **WordPress & Woocommerce**: This integrates with WordPress and Woocommerce allowing the management of orders in Woocommerce.

## Features

- **Mobile Optimization**: Mobile-friendly web application that ensures smooth operation across devices.
- **Easy Check-in**: Direct and simplified check-in process.
- **Attendance Visualisation**: Attendance visualisation to monitor and track members.
